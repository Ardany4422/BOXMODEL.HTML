Box Model

El modelo de caja es un concepto fundamental en la programación y diseño de páginas web. En términos simples, el modelo de caja se refiere a cómo se representan visualmente los elementos HTML en la página. Cada elemento HTML, como un encabezado, un párrafo o una imagen, se representa como una caja rectangular en la página. Esta caja tiene un ancho (width), una altura (height), un margen (margin) alrededor de los bordes, un relleno (padding) dentro de los bordes y un borde (border) opcional alrededor del relleno.
